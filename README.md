# GFM Extended Syntax

## Tables
- `파이프( | )`를 사용하여 열을 표현하고 `( - )`을 사용해 줄을 표현하여 테이블을 만들 수 있다.
  
  ```
  | First Header | Second Header |
  | ------------ | ------------- |
  | content 1 | content 2 |
  | content 3 | content 4 |
  ```
  | First Header | Second Header |
  | ------------ | ------------- |
  | content 1 | content 2 |
  | content 3 | content 4 |

- 첫번째 줄에 ( - )는 길이를 맞출 필요는 없지만 맞추는게 읽기에 좋다.
- `| - |`의 개수와 한 줄에 있는 **content**의 개수는 위의 헤더와 같아야 한다.
